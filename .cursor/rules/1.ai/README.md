# 🤖 AI 공통 규칙 (AI Common Rules)

AI가 모든 프로젝트에서 공통으로 따라야 할 핵심 규칙들입니다.

## 📋 파일 목록

### 1. [ai-role.mdc](./ai-role.mdc) 🔴 필수
**AI의 역할 정의**
- AI가 수행해야 할 역할과 책임
- 풀스택 개발자, Flutter 전문가, 아키텍트 등의 역할
- 작업 프로세스 및 의사결정 원칙
- 성공 기준 및 체크리스트

### 2. [chain-of-thought.mdc](./chain-of-thought.mdc) 🟡 중요
**사고 과정 규칙**
- 복잡한 문제 해결 시 단계적 사고 과정
- CoT 4단계 프로세스 (문제 이해 → 계획 → 구현 → 검증)
- 응답 템플릿 및 작성 가이드
- 복잡도별 적용 방법

### 3. [ai-fit-score.mdc](./ai-fit-score.mdc) 🟢 권장
**AI 룰 품질 평가**
- 규칙 파일의 품질을 정량적으로 평가
- 5개 항목 평가 (명확성, 일관성, 실행 가능성, 위험 회피, AI 최적화)
- 점수화 기준 및 체크리스트

## 🎯 사용 시기

- **프로젝트 시작 전**: ai-role.mdc를 먼저 읽고 역할 이해
- **복잡한 작업 시**: chain-of-thought.mdc를 참고하여 단계적 접근
- **규칙 작성 시**: ai-fit-score.mdc로 품질 검증
